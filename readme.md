1️⃣ What is the difference between var, let, and const?

var: It can be redeclared and reassigned. It is function scoped.

let: It can be reassigned but cannot be redeclared in the same block. It is block scoped.

const: It cannot be reassigned or redeclared. It is block scoped.

2️⃣ What is the spread operator (...)?

==> The spread operator is used to spread the elements of an array or object into separate elements. The spread operator is commonly used for copying arrays and objects.

3️⃣ What is the difference between map(), filter() and forEach()?

map(): It creates a new array by performing operation through each element.

filter(): It creates a new array with elements that match a condition.

forEach(): It executes a function for each element but does not return a new array.


4️⃣ What is an arrow function?

==> An arrow function is a shorter syntax for writing functions in JavaScript introduced in ES6. If the function is one lined, the return keyword is not needed.

Example:

const add = (a, b) => a + b;

5️⃣ What are template literals?

==> Template literals are strings written using backticks (`) that allow embedding variables and expressions inside a string using ${}.
